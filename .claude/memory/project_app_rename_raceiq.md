App renamed to **RaceIQ** (2026-03-27) — browser tab title, server startup logs, etc. The GitHub repo is still `forza-telemetry` but the user-facing brand is RaceIQ.

**Why:** With F1 2025 support added, "Forza Telemetry" no longer accurately described the app. RaceIQ reflects multi-game racing telemetry analysis.

**How to apply:** Use "RaceIQ" when referring to the app in UI text, logs, or user-facing contexts. The repo name `forza-telemetry` may be renamed in the future but is still current.
